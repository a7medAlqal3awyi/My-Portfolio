Welcome to this Flutter project! With a single codebase, you can access this example from mobile, web, and even desktop platforms. 🌍

This project aims to be one of the best examples of Flutter Web, fully open-source, and featuring a range of modern tools and techniques.

## 🤔 How to Use

First of all, configure the Flutter development environments, see <https://flutter.dev/docs/get-started/install>.

- Clone this repository:
`$ git clone https://github.com/a7medAlqal3awyi/My-Portfolio`

- Enter in directory:
`$ cd site`

- Open in your favorite editor (e.g. VSCode):
`$ code .`

- For install dependencies:
`$ flutter pub get`

- Run the app:
`$ flutter run`
